Install AIPA for Discord

Monitor your Discord servers with voice briefings and AI-powered replies.

Discord Bot

AIPA joins your Discord server as a bot. It reads channel messages and can send replies on your behalf.

To connect Discord, you'll need a Discord bot token. Create one at the Discord Developer Portal.

Coming Soon

Discord OAuth integration is coming soon. For now, use a bot token directly in the AIPA app.

Current Setup (Manual)

  • Go to Discord Developer Portal
  • Create a new application and add a bot
  • Enable Message Content Intent under Privileged Gateway Intents
  • Copy the bot token
  • In AIPA iOS app: Settings, add a Discord connection, paste the token
  • Invite the bot to your server using the OAuth2 URL generator

Privacy

Messages are fetched from Discord's API directly to your device. See our Privacy Policy.
