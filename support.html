Getting Started

AIPA connects to your Slack workspace and gives you a voice interface to your channels. To get started:

  • Install the iOS app from TestFlight or the App Store
  • Connect your Slack workspace via Install AIPA for Slack
  • Add your LLM API key (OpenAI required for voice, Anthropic/Gemini optional)
  • Create an agent and link it to channels
  • Tap Connect and start talking

Common Issues

Voice not working

Make sure you have an OpenAI API key configured in Settings. The Realtime API requires an OpenAI key specifically.

Can't read channels

The AIPA bot must be a member of the channels your agent monitors. Invite the bot to the channel in Slack: /invite @AIPA.

Messages not sending

Check that the agent is not in read-only mode (Agent Settings). For sending as yourself, you need AIPA Personal installed with user-level OAuth.
